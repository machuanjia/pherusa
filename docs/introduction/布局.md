# 布局
#### 页面的主题框架，包含导航、头部、banner、主内容
```
# 文件所在 src/layouts

├── src                             // 代码目录
│   ├── layouts                     // 系统布局相关
│   │   ├── banner                  // banner
│   │   ├── header                  // 头部
│   │   ├── main                    // 主布局
│   │   ├── nav                     // 导航
```

```
主布局的使用主要在路由module中使用，详参考路由说明
```
